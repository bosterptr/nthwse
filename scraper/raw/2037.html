    Pomoc

    Wprowadzenie

    To jest overpass turbo, internetowa aplikacja służąca do filtrowania danych OpenStreetMap.

    Poprzez overpass turbo możesz uruchamiać kwerendy Overpass API i analizować zwrócone dane OSM na interaktywnej mapie. Do twojej dyspozycji jest zintegrowany Kreator, dzięki któremu tworzenie kwerend jest bardzo proste.

    Więcej informacji na temat overpass turbo i jak pisać kwerendy Overpass, można znaleźć na wiki OSM.

    Kwerendy Overpass

    Overpass API pozwala wykonywać kwerendy na danych OSM według zadanych kryteriów wyszukiwania. W tym celu posługuje się specjalnie zaprojektowanym językiem kwerend.

    Oprócz zwykłej składni Overpass API overpass turbo oferuje następujące przydatne skróty:

    • {{bbox}} — współrzędne graniczne aktualnego widoku mapy
    • {{center}} — współrzędne środka mapy
    • {{date:…}} — wstawia stempel czasowy ISO 8601 cofnięty o pewien okres (np. "24 hours")
    • {{style:…}} — definiuje arkusz stylów MapCSS
    • Dowolne skróty można definiować, umieszczając {{skrót=wartość}} w treści skryptu.

    Więcej skrótów, dodatkowe informacje o nich i przykłady ich użycia na OSM wiki.

    Udostępnianie

    Możesz wysłać komuś link do kwerendy, nad którą aktualnie pracujesz. Opcję tę umożliwia narzędzie Udostępnij pokazujące link, który możesz wysłać znajomemu lub zamieścić w sieci (zauważ, że inni będą pracowali na własnej kopii kwerendy).

    Zapisz i załaduj

    Możesz także zapisywać i ładować kwerendy. Kilka przykładowych kwerend zostało zapisanych domyślnie. Przejrzyj je, jeśli chcesz zapoznać się pokrótce z możliwościami overpass turbo.

    Skróty klawiszowe:

    • Ctrl+Enter — Uruchom bieżącą kwerendę.
    • Ctrl+I — Uruchom kreator kwerend.
    • Ctrl+O / Ctrl+S — Załaduj (otwórz) / Zapisz kwerendę.
    • Ctrl+H — Otwiera to okno pomocy.

    Legenda mapy

    Linie są wyświetlane jako grube niebieskie linie, wielokąty jako żółte obszary z cienkim niebieskim konturem, POI (węzły z tagami) jako żółte koła z cienkim niebieskim konturem. Koła z czerwonym wypełnieniem to wielokąty lub linie, które są za małe, aby je wyświetlić normalnie. Różowe linie lub kontury oznaczają, że obiekt jest członem przynajmniej jednej (załadowanej) relacji. Przerywane linie oznaczają, że linia lub obszar mają niekompletną geometrię (najczęściej dlatego, że niektóre z ich węzłów nie zostały załadowane).

    Eksport

    Narzędzie Eksportuj pozwala wykonać różne operacje na kwerendzie lub danych przez nią zwracanych.
    Opcje z tym symbolem: polegają na lub odnoszą się do zewnętrznych narzędzi (online).

    Mapa

    Konwertuj obecny widok mapy do statycznego obrazu PNG lub (pełnoekranowej) interaktywnej mapy itd.

    Kwerenda / Dane

    This holds some things you can do with the raw query or data, like converting the query between the various query languages or exporting the data as geoJSON. A very useful option is the possibility to send the query to JOSM.

    Kreator kwerend

    Kreator pomaga tworzyć kwerendy Overpass. Oto kilka przykładów użycia:
    Przykłady
    Drinking Water
    amenity=drinking_water and type:node
    (highway=primary or highway=secondary) and type:way
    tourism=hotel
    tourism=museum in Vienna
    "Drinking Water" in London
    To wyrażenie wyszukiwania jest niezrozumiałe.
    W przypadku ciągów znaków zawierających spacje lub znaki specjalne należy użyć cudzysłowów, ponadto kolejne filtry wyszukiwania muszą być rozdzielone odpowiednimi operatorami logicznymi (and lub or). Więcej informacji można znaleźć w dokumentacji.

    Twoja przeglądarka nie jest obsługiwana :(
