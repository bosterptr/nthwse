Cloning a HID card onto an xEM RFID chip using the Proxmark3 Easy3.0

tthe veii0x

·

Follow

3 min read
·
Mar 13, 2019

I received an xEM RFID chip implant at WOPR Summit 0x0. The xEM RFID chip is from https://store.dangerousthings.com/product/xem/

I was eager to get home and starting playing with the xEM implant. The end result can be seen below.

Prerequisites

Hardware

  • xEM Glass RFID Tag from dangerousthings.com
  • Proxmark3 Easy3.0
  • HID card to be cloned

Software

  • Proxmark Client Software (I used Windows for this demo)

https://github.com/exploitagency/github-proxmark3-standalone-lf-emulator

I downloaded the ZIP file and extracted it.

Using the Windows “Easy Flasher” tool

I did flashed the firmware using this method. This might not be a required for my example below.

https://legacysecuritygroup.com/index.php/categories/9-rfid/7-proxmark-3-emulating-hid-tags-in-standalone-mode

Cloning Process

Plug the Proxmark3 in via USB. Windows should identify the device and assign the Proxmark3 a COM ID, e.g. COM 7.

If the OS doesn’t identify the Proxmark3 you can try pressing and holding the button on the side until the Proxmark3 is recognize by the OS. I had to do this a few times because when I released the button the OS would drop the device.. like a USB dongle had been removed.

Make note of the COM ID, mine was 7.

Launch the Proxmark Client GUI by double-clicking ‘Run Windows GUI Client’

RFID cards will be low frequency, 30-300kHz , or high frequency, 3-30 MHz. To find out the frequency use the ‘lf search’ and ‘hf search’ commands.

Place the HID card on the proxmark3

In the ‘COMMAND TO SEND’ box type ‘lf search’ <Enter>

Locate the TAG ID. The demo card I used is 200507b71a. This is the value that needs to be cloned.

I placed the Proxmark3 directly onto my hand.

Clone the TAG ID by using the ‘lf hid clone’ command.

With the Proxmark3 still on my hand..

Confirm the cloned card by running ‘lf search’.

xEM Chip Specifications

  • 125kHz EM41xx/EM4200/HID/Indala compatible ATA5577 RFID chipset
  • Full datasheet for the Atmel ATA5577 chip
  • Pre-programmed in EM41xx mode with a 40 bit unique ID
  • Encased in 2.1mm by 12mm bioglass with non-toxic epoxy
  • Pre-tested and pre-loaded into sterile injection assembly
  • No “anti-migration” coating means easy removal / replacement
