https://www.wolfandco.com/insight/behind-enemy-lines-pen-tester%E2%80%99s-take-evading-amsi

To run through various techniques learnt I hosted a windows 10 machine running on 192.168.0.103 with real time protection enabled

To test the defender effectiveness I downloaded a reverse shell script from https://github.com/samratashok/nishang/blob/master/Shells/Invoke-PowerShellTcpOneLine.ps1.

I first scanned the script using virus total to check if it is malicious and to my surprise it was declared as clean by virus total

However the challenge came while executing the powershell script.
