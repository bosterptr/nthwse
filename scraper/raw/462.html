Our Service

We collect a lot of samples reported by sandboxes and malware analysis services. Among all this data we can find both malicious software and legitimate software. For both types of data we have interesting information such as contacted domains, files written in the system or processes executed by malware sample.

We provide this data as datasets, useful for big data in graphical network visualization and machine learning. In the same way, these data can be used by companies and researchers as threat intelligence feed.
