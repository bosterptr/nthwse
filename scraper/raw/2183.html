Hacktools, is a web extension facilitating your web application penetration tests, it includes cheat sheets as well as all the tools used during a test such as XSS payloads, Reverse shells to test your web application.

About this extension
Hacktools make your penetration testing procedure faster by providing most of the tools avalaible and dynamically generated in one click!
It is accessible either in pop up mode or in a whole tab in the Devtools part of the browser with F12.
